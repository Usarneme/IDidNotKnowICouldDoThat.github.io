I share my code!

Studying software development, I've been learning a lot! In order to ensure I understand new concepts I have been making projects to try and solidify them in my mind and also create something that may be of use to others. I try and share anything I do that could be useful to others (and myself later on) in my various GitHub repositories.

Recently I've been studying Facebook's React.js library for building front-end components quickly and in a way that is easy to understand when projects scale up. With React and Redux one can manage large sets of dependencies more simply and create template objects that can be reused again and again in your code with ease.

In order to solidify my knowledge and share what I've learned of React, node.js, npm, the Create-React-App script, CSS preprocessing using SASS, and creating a set of instructions to manage project dependencies with a package.json file, I created a Bash shell script to automate the deployment of a web development environment.

With one (if you have all the dependencies already installed) or just a few simple commands you can get this development environment in place with all of the boilerplate template files and partials you'll need to quickly build a Progressive Web App utilizing the bleeding edge of the Web Development world's toolset.
